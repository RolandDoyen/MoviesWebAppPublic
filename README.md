# Movies Webapp (Azure)
This is the **Azure Web Apps** version of the Movies frontend.  
It allows viewing, creating, editing, and deleting movies via the Movies API.

## Live Demo
[Movies Webapp on Azure](https://movies-rd.azurewebsites.net)

## Notes
- No authentication token is required; CORS is handled by the API.
- Use this repo to deploy or test the webapp on Azure.
