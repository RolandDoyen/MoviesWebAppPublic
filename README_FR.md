# Movies Webapp (Azure)
Ceci est la version **Azure Web Apps** du frontend Movies.  
Elle permet de visualiser, créer, modifier et supprimer des films via l’API Movies.

## Démo en Ligne
[Movies Webapp sur Azure](https://movies-rd.azurewebsites.net)

## Remarques
- Aucun token d’authentification n’est requis ; le CORS est géré par l’API.  
- Utilisez ce repo pour déployer ou tester la webapp sur Azure.
